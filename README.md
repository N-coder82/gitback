# gitback
A git-based backup maker/manager
## How to install:
Run this command:
```
/bin/bash <(curl -s https://raw.githubusercontent.com/N-coder82/gitback/main/install.sh)
```
## How to uninstall:

Run this command:

```
/bin/bash <(curl -s https://raw.githubusercontent.com/N-coder82/gitback/main/uninstall.sh)
```
